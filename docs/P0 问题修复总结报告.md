### P0-1: 修复测试脚本 CSRF 处理

**问题描述**:
- 集成测试脚本无法正确处理 CSRF token
- 导致所有需要认证的 API 测试失败
- 集成测试通过率为 0%

**修复方案**:
1. 在登录流程中添加 CSRF token 提取逻辑
2. 使用正则表达式从 HTML 中提取 CSRF token
3. 在登录请求中包含 CSRF token

**修复代码**:
```python
def login(self):
    """登录获取 session 和 CSRF token"""
    # 1. 获取登录页面
    login_page = self.session.get(f"{BASE_URL}/login")
    
    # 2. 提取 CSRF token
    import re
    match = re.search(r'name="csrf_token"\s+value="([^"]+)"', login_page.text)
    if match:
        csrf_token = match.group(1)
    
    # 3. 使用 CSRF token 登录
    login_data = {
        'username': 'admin',
        'password': 'admin123',
        'csrf_token': csrf_token  # 添加 CSRF token
    }
    response = self.session.post(f"{BASE_URL}/login", data=login_data)
```

**修复结果**:
- ✅ CSRF token 成功提取
- ✅ 登录流程正常工作
- ⚠️ 后端返回 500 错误（需进一步排查）
