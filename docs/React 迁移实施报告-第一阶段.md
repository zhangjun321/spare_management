## ⚠️ 遇到的问题和解决方案

### 问题 1：CSRF Token 处理

**问题描述：** Flask 的 CSRF 保护导致 API 请求失败

**解决方案：**
```javascript
// 从 cookie 获取 CSRF token
const csrfToken = document.cookie
  .split('; ')
  .find(row => row.startsWith('csrf_token='))
  ?.split('=')[1]

if (csrfToken) {
  config.headers['X-CSRFToken'] = csrfToken
}
```

### 问题 2：Chart.js 在 React 中的使用

**问题描述：** Chart.js 需要在 DOM 渲染完成后初始化

**解决方案：**
- 使用 `react-chartjs-2` 封装
- 自动处理生命周期
- 响应式更新

### 问题 3：侧边栏响应式

**问题描述：** 移动端和桌面端的侧边栏行为不同

**解决方案：**
```javascript
// 移动端遮罩层
{!sidebarCollapsed && (
  <div 
    className="d-lg-none fixed-top w-100 h-100"
    style={{ background: 'rgba(0,0,0,0.5)', zIndex: 998 }}
    onClick={toggleSidebar}
  />
)}
```
