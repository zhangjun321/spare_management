# 仓库管理 V3 - 全新前端使用说明

## 📋 概述

仓库管理 V3 是一个全新的、现代化的智能仓库管理系统，采用先进的 UI 设计理念和强大的功能模块，帮助用户高效管理仓库运营。

## ✨ 核心特性

### 1. 现代化 UI 设计
- **专业配色方案**：采用蓝紫色系渐变，营造科技感
- **响应式布局**：完美适配桌面、平板和手机
- **侧边栏导航**：清晰的层级结构，快速切换功能
- **卡片式设计**：信息模块化，易于阅读和操作

### 2. 智能仪表盘
- **实时统计数据**：仓库总数、货位总数、库存 SKU、预警数量
- **库存趋势图表**：可视化展示出入库趋势
- **仓库利用率**：饼图显示仓库使用情况
- **库存预警**：实时显示库存异常情况

### 3. 仓库管理
- **仓库列表**：展示所有仓库的详细信息
- **快速操作**：新建、查看、编辑、删除仓库
- **状态监控**：实时显示仓库状态和使用率
- **AI 启用标识**：标识 AI 功能启用状态

### 4. 库存管理
- **库存列表**：查看所有仓库的库存情况
- **状态标识**：正常、低库存、缺货预警
- **库存调整**：快速调整库存数量
- **仓库筛选**：按仓库筛选库存

### 5. 货位管理
- **可视化展示**：卡片式展示货位状态
- **状态分类**：已占用、已预留、空闲
- **货位类型**：显示货位类型和规格

### 6. AI 智能分析
- **备件数据分析**：AI 分析备件特性，提供存储建议
- **货位智能推荐**：基于备件特性推荐最优货位
- **拣货路径优化**：AI 优化拣货路径，提高效率
- **需求预测**：预测未来需求，辅助决策

## 🚀 快速开始

### 访问系统
1. 打开浏览器，访问 `http://127.0.0.1:5000/warehouse-v3/`
2. 使用管理员账户登录（默认：admin / admin123）
3. 进入全新的仓库管理界面

### 主要功能操作

#### 查看仪表盘
- 默认页面显示关键指标和图表
- 点击预警信息可快速跳转到相关仓库

#### 管理仓库
1. 点击左侧菜单"仓库管理"
2. 点击"新建仓库"创建新仓库
3. 使用搜索框快速查找仓库
4. 点击操作按钮进行查看、编辑、删除

#### 管理库存
1. 点击左侧菜单"库存管理"
2. 选择仓库筛选库存
3. 查看库存状态和预警
4. 点击调整按钮修改库存

#### 使用 AI 功能
1. 点击左侧菜单"AI 分析"
2. 选择"备件数据分析"或"货位智能推荐"
3. 点击开始按钮执行分析
4. 查看 AI 分析结果和建议

## 📊 功能模块详解

### 仪表盘模块
**统计卡片**
- 仓库总数：显示系统中文仓库数量
- 货位总数：显示所有可用货位数量
- 库存 SKU：显示当前库存的 SKU 数量
- 预警数量：显示需要关注的库存预警数量

**库存趋势图**
- 显示最近 7/30/90 天的出入库趋势
- 支持切换时间周期
- 直观展示库存变化

**仓库利用率**
- 饼图显示仓库整体使用情况
- 已使用 vs 空闲货位对比

**库存预警**
- 实时显示库存不足、超储等预警
- 点击可直接跳转到相关仓库处理

### 仓库管理模块
**仓库列表功能**
- 编码、名称、类型一目了然
- 使用率进度条直观显示
- 状态徽章显示运营状态
- AI 启用标识显示智能化程度

**操作功能**
- 新建仓库：填写表单快速创建
- 查看仓库：查看详细信息和统计
- 编辑仓库：修改仓库配置
- 删除仓库：删除不需要的仓库

### 库存管理模块
**库存列表**
- 备件编码和名称
- 所在仓库和货位
- 库存数量和单位
- 库存状态（正常/低库存/缺货）

**库存调整**
- 盘盈盘亏处理
- 库存调拨
- 库存状态变更

### 货位管理模块
**可视化展示**
- 卡片式展示每个货位
- 颜色区分使用状态
- 显示货位编码和类型

**状态管理**
- 绿色：已占用
- 黄色：已预留
- 灰色：空闲

### AI 分析模块
**备件数据分析**
- 分析备件物理特性
- 识别存储要求
- 提供分类建议

**货位智能推荐**
- 基于备件特性
- 考虑仓库布局
- 提供最优货位建议

**拣货路径优化**
- 分析拣货清单
- 计算最优路径
- 提高拣货效率

**需求预测**
- 分析历史数据
- 预测未来需求
- 辅助采购决策
