## 二、新增表 DDL

### DB-01：调拨单主表 `transfer_order`

```sql
-- MySQL 8.0 版本
CREATE TABLE `transfer_order` (
    `id`                  INT            NOT NULL AUTO_INCREMENT COMMENT '调拨单ID',
    `order_no`            VARCHAR(50)    NOT NULL UNIQUE         COMMENT '调拨单号，格式：DB-YYYYMMDD-NNNN',
    `from_warehouse_id`   INT            NOT NULL                COMMENT '调出仓库ID',
    `to_warehouse_id`     INT            NOT NULL                COMMENT '调入仓库ID',
    `from_location_id`    INT            NULL                    COMMENT '调出货位ID',
    `to_location_id`      INT            NULL                    COMMENT '调入货位ID',
    `transfer_type`       VARCHAR(20)    NOT NULL DEFAULT 'normal' COMMENT '调拨类型: normal-普通/emergency-紧急/balance-平衡调拨',
    `status`              VARCHAR(20)    NOT NULL DEFAULT 'draft' COMMENT '状态: draft/pending/approved/in_transit/completed/cancelled',
    `priority`            TINYINT        NOT NULL DEFAULT 2      COMMENT '优先级: 1-紧急 2-普通 3-低',
    `approval_required`   TINYINT(1)     NOT NULL DEFAULT 1      COMMENT '是否需要审批',
    `reason`              TEXT           NULL                    COMMENT '调拨原因',
    `remark`              TEXT           NULL                    COMMENT '备注',
    -- 关联单据
    `outbound_order_id`   INT            NULL                    COMMENT '关联出库单ID',
    `inbound_order_id`    INT            NULL                    COMMENT '关联入库单ID',
    -- 审批信息
    `approved_by`         INT            NULL                    COMMENT '审批人ID',
    `approved_at`         DATETIME       NULL                    COMMENT '审批时间',
    `approval_comment`    VARCHAR(500)   NULL                    COMMENT '审批意见',
    -- 执行信息
    `estimated_arrival`   DATE           NULL                    COMMENT '预计到达日期',
    `actual_arrival`      DATETIME       NULL                    COMMENT '实际到达时间',
    `carrier`             VARCHAR(100)   NULL                    COMMENT '承运方',
    `tracking_no`         VARCHAR(100)   NULL                    COMMENT '运单号',
    -- 审计字段
    `created_by`          INT            NOT NULL                COMMENT '创建人ID',
    `confirmed_by`        INT            NULL                    COMMENT '确认人ID',
    `confirmed_at`        DATETIME       NULL                    COMMENT '确认时间',
    `completed_at`        DATETIME       NULL                    COMMENT '完成时间',
    `cancelled_at`        DATETIME       NULL                    COMMENT '取消时间',
    `cancel_reason`       VARCHAR(500)   NULL                    COMMENT '取消原因',
    `created_at`          DATETIME       NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `updated_at`          DATETIME       NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    PRIMARY KEY (`id`),
    INDEX `idx_from_warehouse` (`from_warehouse_id`),
    INDEX `idx_to_warehouse`   (`to_warehouse_id`),
    INDEX `idx_status`         (`status`),
    INDEX `idx_created_at`     (`created_at`),
    CONSTRAINT `fk_transfer_from_wh`   FOREIGN KEY (`from_warehouse_id`) REFERENCES `warehouse`(`id`),
    CONSTRAINT `fk_transfer_to_wh`     FOREIGN KEY (`to_warehouse_id`)   REFERENCES `warehouse`(`id`),
    CONSTRAINT `fk_transfer_created`   FOREIGN KEY (`created_by`)        REFERENCES `user`(`id`),
    CONSTRAINT `fk_transfer_approved`  FOREIGN KEY (`approved_by`)       REFERENCES `user`(`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='调拨单主表';
```

## 八、数据迁移注意事项

1. **`abc_class` 初始值**：新增字段默认为 NULL，需在系统首次运行 ABC 分析任务后自动填充，不需要手动初始化。

2. **`fifo_sequence` 初始值**：现有 `inventory_record` 记录的 `fifo_sequence` 默认为 0，运维团队可在迁移后执行以下 SQL 按入库时间重新排序：

```sql
-- 为现有记录按创建时间设置FIFO序号
UPDATE inventory_record ir
JOIN (
    SELECT id,
           ROW_NUMBER() OVER (PARTITION BY spare_part_id, warehouse_id ORDER BY created_at) AS rn
    FROM inventory_record
) ranked ON ir.id = ranked.id
SET ir.fifo_sequence = ranked.rn;
```

3. **调拨单与现有出入库单联动**：`transfer_order.outbound_order_id` 和 `inbound_order_id` 存储关联单据 ID，调拨流程完成后由业务代码写入，无需迁移时处理。

4. **SQLite 外键约束**：SQLite 默认不强制外键，若需启用：

```python
# app/extensions.py 或应用初始化时
from sqlalchemy import event
from sqlalchemy.engine import Engine
import sqlite3

@event.listens_for(Engine, "connect")
def set_sqlite_pragma(dbapi_connection, connection_record):
    if isinstance(dbapi_connection, sqlite3.Connection):
        cursor = dbapi_connection.cursor()
        cursor.execute("PRAGMA foreign_keys=ON")
        cursor.close()
```
