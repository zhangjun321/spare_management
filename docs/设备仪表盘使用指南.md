# 设备仪表盘使用指南

## 🎉 功能已就绪！

基于 **Leaflet + OpenStreetMap** 的设备仪表盘已开发完成，**无需任何 API Key**，开箱即用！

---

## 🚀 立即访问

### 方式一：通过导航菜单
1. 登录系统
2. 点击左侧导航菜单的 **"设备管理"**
3. 点击子菜单 **"设备仪表盘"**

### 方式二：直接访问 URL
```
http://localhost:5000/equipment/dashboard
```

---

## 📊 功能概览

### 1. 设备统计卡片
- **设备总数**: 显示系统中的所有设备数量
- **运行中**: 绿色标记，正常使用中的设备
- **停机**: 灰色标记，暂停使用的设备
- **维修中**: 橙色标记，正在维修的设备
- **已报废**: 红色标记，已报废的设备
- **有位置设备**: 已录入经纬度坐标的设备数量

### 2. 设备分布地图
- 显示中国地图（基于 OpenStreetMap）
- 彩色圆点标记设备位置
- 不同颜色表示不同状态
- 支持缩放、平移
- 自动调整视野显示所有设备

### 3. 图例说明
- 🟢 绿色圆点：运行中
- ⚫ 灰色圆点：停机
- 🟠 橙色圆点：维修中
- 🔴 红色圆点：已报废

### 4. 设备系列统计
- FS 系列：固定式/快速通道
- MT 系列：车载移动式
- CS 系列：小型车辆检查
- MB 系列：组合移动式
- FG 系列：固定式集装箱

### 5. 设备列表
- 右侧显示所有有位置的设备
- 点击列表项可快速定位到地图上的设备
- 支持滚动查看所有设备

### 6. 设备详情弹窗
- 点击地图上的标记查看设备详情
- 显示信息：
  - 设备名称、编号
  - 型号、系列
  - 状态
  - 安装位置
  - 使用部门
  - 责任人
  - 经纬度坐标
- 点击"查看详情"按钮跳转到设备详情页

---

## 🗺️ 地图操作

### 基本操作
- **拖动**: 鼠标左键按住拖动地图
- **放大/缩小**: 鼠标滚轮滚动
- **双击放大**: 双击地图任意位置
- **缩放控件**: 左上角的 +/- 按钮

### 查看设备
- **点击标记**: 查看设备简要信息
- **点击列表**: 自动定位并显示设备详情
- **飞行定位**: 从列表点击后有平滑飞行动画

### 调整视野
- **刷新地图**: 点击右上角刷新按钮，自动调整到最佳视野
- **手动缩放**: 使用滚轮或缩放控件调整

## 🔧 技术实现

### 前端技术栈
- **Leaflet.js**: 开源地图库（无需 API Key）
- **OpenStreetMap**: 开源地图数据
- **Bootstrap 5**: UI 框架
- **原生 JavaScript**: 无需额外依赖

### 后端 API
- **GET /equipment/dashboard**: 仪表盘页面
- **GET /equipment/api/locations**: 设备位置数据接口

### 数据库字段
```sql
ALTER TABLE equipment ADD COLUMN latitude DECIMAL(10, 8) COMMENT '纬度';
ALTER TABLE equipment ADD COLUMN longitude DECIMAL(11, 8) COMMENT '经度';
ALTER TABLE equipment ADD COLUMN map_address VARCHAR(500) COMMENT '地图地址';
```

## 🎨 自定义配置

### 修改地图中心点
编辑 `dashboard_leaflet.html`，找到：
```javascript
map = L.map('mapContainer').setView([37.564716, 104.066544], 4);
```
修改经纬度和缩放级别：
- `[37.564716, 104.066544]`: 中心点坐标（中国中心）
- `4`: 缩放级别（数字越大越详细）

### 修改标记颜色
编辑 `dashboard_leaflet.html`，找到：
```javascript
const markerColors = {
    'green': '#198754',   // 运行中
    'gray': '#6c757d',    // 停机
    'orange': '#ffc107',  // 维修中
    'red': '#dc3545'      // 已报废
};
```
修改颜色值即可。

### 修改标记大小
编辑 `dashboard_leaflet.html`，找到：
```javascript
const marker = L.circleMarker(latlng, {
    radius: 10,  // 修改此值调整标记大小
    ...
});
```

## 🆚 与百度地图对比

| 特性 | Leaflet (当前) | 百度地图 |
|------|---------------|----------|
| **API Key** | ❌ 无需 | ✅ 需要申请 |
| **费用** | ✅ 完全免费 | ⚠️ 超出配额后收费 |
| **国内精度** | ⭐⭐⭐ 一般 | ⭐⭐⭐⭐⭐ 精确 |
| **国际覆盖** | ⭐⭐⭐⭐⭐ 优秀 | ⭐⭐⭐ 一般 |
| **地理编码** | ⚠️ 需额外服务 | ✅ 内置支持 |
| **街景** | ❌ 不支持 | ✅ 支持 |
| **路线规划** | ⚠️ 需插件 | ✅ 内置支持 |
| **文档** | ⭐⭐⭐⭐ 英文为主 | ⭐⭐⭐⭐⭐ 中文完善 |

### 建议
- ✅ **当前方案（Leaflet）**: 适合快速部署、无需审批、完全免费
- 🔄 **可选升级（百度地图）**: 如果需要地理编码、街景、路线规划等高级功能
