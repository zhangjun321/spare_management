# 列显示配置功能说明

## 📋 功能概述

备件列表页面新增了**列显示配置**功能，用户可以根据需要自定义显示哪些列，提升工作效率和用户体验。

---

## 🎯 使用方法

### 1. 打开列配置菜单

点击列表页面右上角的 **"列显示"** 按钮（图标为 📊 列），会弹出列配置下拉菜单。

### 2. 选择显示的列

**快速操作**：
- ✅ **全部列**：勾选/取消勾选，一键选择/取消所有列
- ✅ **单个列**：单独勾选/取消某个列

**可配置的列**：
1. 缩略图
2. 备件代码
3. 名称
4. 规格型号
5. 分类
6. 供应商
7. 当前库存
8. 库存状态
9. 单价
10. 单位
11. 状态（启用/停用）
12. 操作（查看/编辑/删除）

**注意**：复选框列（第一列）始终显示，无法隐藏。

### 3. 应用设置

选择完成后，点击下拉菜单底部的按钮：
- 🔵 **应用**：保存并应用当前选择
- ⚪ **重置**：恢复默认设置（显示所有列）

---

## 💡 功能特点

### 1. 智能联动

- **全部列复选框**：
  - 勾选 → 自动选中所有列
  - 取消勾选 → 自动取消所有列
  - 手动选择/取消列时，自动更新状态

- **状态同步**：
  - 所有列都选中时，"全部列"自动勾选
  - 取消任一列，"全部列"自动取消

### 2. 记忆功能

- **自动保存**：应用设置后自动保存到浏览器本地存储
- **恢复设置**：下次访问时自动恢复上次的列配置
- **跨会话**：即使关闭浏览器，设置也会保留

### 3. 即时生效

- 点击"应用"按钮后立即生效
- 无需刷新页面
- 自动关闭下拉菜单

## 🎨 使用场景

### 场景 1：快速查看关键信息

**需求**：只查看备件代码、名称和库存

**操作**：
1. 点击"列显示"按钮
2. 取消勾选"全部列"
3. 勾选：备件代码、名称、当前库存
4. 点击"应用"

**效果**：只显示这 3 列，页面更简洁

---

### 场景 2：财务分析

**需求**：重点关注价格信息

**操作**：
1. 点击"列显示"按钮
2. 确保勾选：单价、单位、供应商
3. 可选：取消勾选缩略图、规格型号等
4. 点击"应用"

**效果**：突出显示财务相关信息

---

### 场景 3：库存管理

**需求**：专注库存数据

**操作**：
1. 点击"列显示"按钮
2. 确保勾选：当前库存、库存状态、最低库存（如有）
3. 可选：取消勾选操作列
4. 点击"应用"

**效果**：便于库存盘点和管理

---

### 场景 4：演示和展示

**需求**：展示完整信息

**操作**：
1. 点击"列显示"按钮
2. 勾选"全部列"
3. 点击"应用"

**效果**：显示所有列，信息完整

## 🔧 技术实现

### HTML 结构

```html
<div class="dropdown">
    <button class="btn btn-outline-secondary dropdown-toggle">
        <i class="fas fa-columns"></i> 列显示
    </button>
    <ul class="dropdown-menu">
        <li><h6 class="dropdown-header">选择显示的列</h6></li>
        <li>
            <div class="dropdown-item">
                <div class="form-check">
                    <input type="checkbox" value="all"> 全部列
                </div>
            </div>
        </li>
        <!-- 各个列的复选框 -->
    </ul>
</div>
```

### JavaScript 功能

1. **复选框联动**
   - 监听"全部列"复选框变化
   - 监听单个列复选框变化
   - 自动更新关联状态

2. **列显示控制**
   - 获取选中的列
   - 遍历表格的 th 和 td
   - 根据 data-column 属性显示/隐藏

3. **本地存储**
   - 使用 localStorage 保存配置
   - 页面加载时恢复配置
   - 支持重置清除配置

### 数据属性

```html
<th data-column="part_code">备件代码</th>
<td data-column="part_code">SKFX-A-001</td>
```

通过 `data-column` 属性关联表头和单元格

## ⚠️ 注意事项

### 1. 必显列

以下列始终显示，无法隐藏：
- **复选框列**：用于批量操作
- 如需要隐藏，请修改代码

### 2. 浏览器兼容性

- 需要支持 localStorage 的现代浏览器
- Chrome、Firefox、Edge、Safari 均支持
- IE10 及以下版本不支持

### 3. 清除配置

**方法 1**：点击"重置"按钮
**方法 2**：清除浏览器缓存
**方法 3**：打开浏览器开发者工具，执行：
```javascript
localStorage.removeItem('sparePartsColumns');
location.reload();
```

### 4. 性能考虑

- 列数过多时，隐藏部分列可提升渲染性能
- 建议只显示必要的 5-8 列
- 卡片视图不受列配置影响

## 🔄 与其他功能配合

### 1. 视图切换

- 列配置仅影响**列表视图**
- **卡片视图**不受影响
- 可以自由组合使用

### 2. 批量导出

- 列配置不影响导出内容
- 导出始终包含所有字段
- 可放心配置列显示

### 3. 搜索筛选

- 列配置与搜索功能独立
- 搜索不受列显示影响
- 可以先筛选再配置列
