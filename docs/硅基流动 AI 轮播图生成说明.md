## 🛠️ 技术实现

### API 信息
- **平台**：硅基流动 (SiliconFlow)
- **模型**：Kwai-Kolors/Kolors
- **API 地址**：https://api.siliconflow.cn/v1/images/generations
- **图片规格**：1920x600
- **质量**：HD 高清

### 脚本文件
1. **`scripts/generate_carousel_images.py`** - 主生成脚本（生成所有 48 张图片）
2. **`scripts/generate_remaining_images.py`** - 补充生成脚本（生成剩余图片）
3. **`scripts/batch_update_carousel.py`** - 批量更新轮播图组件脚本

## 🚀 使用方法

### 重新生成图片（如果需要）

```bash
# 1. 确保已安装依赖

# 2. 运行生成脚本
cd scripts
python generate_carousel_images.py
```

### 更新轮播图组件

```bash
# 批量更新所有轮播图组件，使用本地图片
python batch_update_carousel.py
```

## ⚠️ 注意事项

### API 密钥安全
- **重要**：API 密钥包含敏感信息，不应提交到版本控制系统
- 建议使用环境变量或配置文件管理 API 密钥
- 当前密钥已包含在脚本中，生产环境请替换

### API 限流处理
- 硅基流动 API 有请求频率限制
- 脚本已实现自动重试机制（最多 5 次）
- 遇到 429 错误时会指数退避等待（15s, 30s, 45s...）
- 建议批量生成时设置适当的延迟

### 图片生成时间
- 每张图片生成约需 5-30 秒（取决于 API 响应速度）
- 48 张图片总计约需 15-20 分钟
- 包含重试机制，实际时间可能更长
