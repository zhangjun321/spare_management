执行 SQL 更新语句，批量修复所有记录的库存状态：

```sql
UPDATE spare_part 
SET stock_status = CASE
    WHEN current_stock <= min_stock THEN 'low'
    WHEN max_stock IS NOT NULL AND current_stock >= max_stock THEN 'overstocked'
    ELSE 'normal'
END
```

## 📝 库存状态规则

### 判断逻辑

```python
def update_stock_status(self):
    """更新库存状态"""
    if self.current_stock <= self.min_stock:
        self.stock_status = 'low'  # 低库存/缺货
    elif self.max_stock and self.current_stock >= self.max_stock:
        self.stock_status = 'overstocked'  # 超储
    else:
        self.stock_status = 'normal'  # 正常
```

### 状态定义

| 状态 | 条件 | 说明 | 图标 | 颜色 |
|------|------|------|------|------|
| normal | current_stock > min_stock AND current_stock < max_stock | 库存正常 | ✓ | 绿色 |
| low | current_stock <= min_stock | 低库存/缺货 | ⚠️ | 红色 |
| overstocked | current_stock >= max_stock | 超储 | ⚠️ | 黄色 |

## 🎯 后续改进

### 1. 添加数据库触发器

在数据库层面自动更新状态：

```sql
DELIMITER $$
CREATE TRIGGER update_stock_status_trigger
BEFORE INSERT OR UPDATE ON spare_part
FOR EACH ROW
BEGIN
    IF NEW.current_stock <= NEW.min_stock THEN
        SET NEW.stock_status = 'low';
    ELSEIF NEW.max_stock IS NOT NULL AND NEW.current_stock >= NEW.max_stock THEN
        SET NEW.stock_status = 'overstocked';
    ELSE
        SET NEW.stock_status = 'normal';
    END IF;
END$$
DELIMITER ;
```

### 2. 定期检查和修复

创建定时任务，每天检查并修复库存状态：

```python
def check_and_fix_stock_status():
    """检查并修复所有备件的库存状态"""
    parts = SparePart.query.all()
    fixed_count = 0
    
    for part in parts:
        old_status = part.stock_status
        part.update_stock_status()
        
        if part.stock_status != old_status:
            fixed_count += 1
    
    db.session.commit()
    return fixed_count
```

### 3. 添加数据验证

在模型中添加验证方法：

```python
@staticmethod
def validate_all_stock_status():
    """验证所有备件的库存状态是否正确"""
    parts = SparePart.query.all()
    errors = []
    
    for part in parts:
        expected_status = 'normal'
        if part.current_stock <= part.min_stock:
            expected_status = 'low'
        elif part.max_stock and part.current_stock >= part.max_stock:
            expected_status = 'overstocked'
        
        if part.stock_status != expected_status:
            errors.append({
                'id': part.id,
                'part_code': part.part_code,
                'current_stock': part.current_stock,
                'expected_status': expected_status,
                'actual_status': part.stock_status
            })
    
    return errors
```
