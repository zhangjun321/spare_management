## 🔍 详细产品型号及参数

### 1️⃣ FS 系列 - 固定式/快速通道检查系统

**产品定位**：高速通过的固定式 X 射线检查系统，适用于高流量检查场景

#### ✅ 典型型号

| 型号 | 扫描方式 | 通过率 | 成像技术 | 特点 | 应用场景 |
|------|----------|--------|----------|------|----------|
| **FS6000** | X 射线透射 | 120-200 辆/小时 | 双能成像（IDE） | 车辆不停通过（≤15 km/h） | 高速公路卡口、港口、物流园 |
| **FS6000MF** | X 射线透射 | 120-200 辆/小时 | 双能成像（IDE） | 多视角成像 | 海关集中查验场 |
| **FS0200BX** | 背散射成像 | 高速通过 | 背散射成像 | 低剂量，驾驶员可在车内 | 城市出入口、大型活动安保 |

#### 📋 核心参数指标

| 参数项 | 数值/描述 |
|--------|-----------|
| **扫描方式** | X 射线透射 / 背散射成像 |
| **通过方式** | 车辆不停通过（≤15 km/h） |
| **检查效率** | 120-200 辆/小时 |
| **成像技术** | 双能成像（IDE）、背散射（FS0200BX） |
| **安全性** | 低剂量设计（驾驶员可在车内） |
| **视角** | 单视角 / 双视角 / 三视角 |
| **典型应用** | 高速公路卡口、港口/物流园、海关集中查验 |

---

### 2️⃣ MT 系列 - 车载移动式检查系统

**产品定位**：高机动性车载式 X 射线检查系统，即到即检

#### ✅ 典型型号

| 型号 | 加速器能量 | 扫描模式 | 特点 | 应用场景 |
|------|------------|----------|------|----------|
| **MT1213LT / LT(D)** | 4-6 MeV | 车载：20-25 辆/小时<br>快检：最高 150 辆/小时 | 整车集成、无需土建 | 临时检查点、边境巡检 |
| **MT1213LH / LH(T)** | 6/3 MeV (双能) | 车载：20-25 辆/小时<br>快检：最高 150 辆/小时 | 双能物质识别 | 应急安检、移动执法 |
| **MT1213LC** | 未公开 | 未公开 | 紧凑型设计 | 城市巡逻 |
| **MT1213DE / DE(T)** | 6 MeV | 未公开 | 出口型、高配置 | 国际项目 |

#### 📋 核心参数指标

| 参数项 | 数值/描述 |
|--------|-----------|
| **加速器能量** | 4-6 MeV 或 6/3 MeV（双能） |
| **扫描模式** | 车载扫描：20-25 辆/小时<br>快检模式：最高 150 辆/小时 |
| **特点** | 整车集成（可直接开到现场）<br>无需土建<br>支持极端气候运行 |
| **成像技术** | 双能物质识别 |
| **典型应用** | 临时检查点、边境巡检、应急安检、移动执法 |

### 3️⃣ CS 系列 - 小型车辆检查系统

**产品定位**：针对小型车辆、轿车的顶视 X 射线检查系统

| **CS0300TS / CS1000TS** | 顶视 X 射线成像 | 标准：约 30 辆/小时<br>快检：最高 150 辆/小时 | 传送带拖动 | 增强型、高分辨率 | 大型活动安检 |
| **CS0200T / CS0200M** | 顶视 X 射线成像 | 未公开 | 手动/自动 | 紧凑型 | 重要场所入口 |
| **CS0300T(BX)** | 顶视 + 背散射 | 未公开 | 手动/自动 | 双模式检测 | 海关车辆检查 |

## 💡 设备管理系统数据库设计

### 核心表结构

#### 1. 设备主表 (`equipment`)

| 字段名 | 类型 | 说明 | 必填 |
|--------|------|------|------|
| `id` | INT | 主键 | ✅ |
| `equipment_code` | VARCHAR(50) | 设备编号（如：SEC-MT-2024-001） | ✅ |
| `model` | VARCHAR(100) | 设备型号（如：MT1213LT） | ✅ |
| `series` | VARCHAR(20) | 设备系列（FS/MT/CS/MB/FG/PB/RF） | ✅ |
| `manufacturer` | VARCHAR(200) | 生产厂家（同方威视） | ✅ |
| `serial_number` | VARCHAR(100) | 出厂编号 | ✅ |
| `purchase_date` | DATE | 采购日期 | ✅ |
| `install_date` | DATE | 安装日期 | ✅ |
| `install_location` | VARCHAR(500) | 安装地点 | ✅ |
| `status` | VARCHAR(20) | 设备状态（RUNNING/STOPPED/MAINTENANCE/SCRAPPED） | ✅ |
| `department` | VARCHAR(200) | 使用部门 | ✅ |
| `responsible_person` | VARCHAR(100) | 责任人 | ✅ |
| `created_at` | DATETIME | 创建时间 | ✅ |
| `updated_at` | DATETIME | 更新时间 | ✅ |

#### 2. 技术参数表 (`equipment_specs`)

| 字段名 | 类型 | 说明 | 必填 |
|--------|------|------|------|
| `id` | INT | 主键 | ✅ |
| `equipment_id` | INT | 设备 ID（外键） | ✅ |
| `radiation_source` | VARCHAR(100) | 射线源类型（电子直线加速器/X 射线管） | ✅ |
| `energy` | VARCHAR(50) | 能量（如：6 MeV、4-6 MeV） | ✅ |
| `penetration` | VARCHAR(50) | 穿透力（如：320mm 钢板） | ✅ |
| `resolution` | VARCHAR(50) | 分辨率（如：3.5mm） | ✅ |
| `check_speed` | VARCHAR(50) | 检查速度（如：120-200 辆/小时） | ✅ |
| `channel_size` | VARCHAR(100) | 通道尺寸（如：4.5m×4.5m） | ✅ |
| `imaging_mode` | VARCHAR(100) | 成像方式（双能/CT/顶视/背散射） | ✅ |
| `shielding_mode` | VARCHAR(100) | 屏蔽方式（自屏蔽/混凝土） | ✅ |
| `other_specs` | JSON | 其他参数 | ❌ |

#### 3. 维护记录表 (`maintenance_records`)

| 字段名 | 类型 | 说明 | 必填 |
|--------|------|------|------|
| `id` | INT | 主键 | ✅ |
| `equipment_id` | INT | 设备 ID（外键） | ✅ |
| `maintenance_type` | VARCHAR(50) | 维护类型（保养/维修/检定） | ✅ |
| `maintenance_date` | DATE | 维护日期 | ✅ |
| `description` | TEXT | 维护内容 | ✅ |
| `result` | TEXT | 维护结果 | ✅ |
| `cost` | DECIMAL(10,2) | 维护费用 | ❌ |
| `technician` | VARCHAR(100) | 技术人员 | ✅ |
| `next_maintenance_date` | DATE | 下次维护日期 | ❌ |
| `created_at` | DATETIME | 创建时间 | ✅ |
