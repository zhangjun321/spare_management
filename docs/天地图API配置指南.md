# 天地图 API 配置指南

## 为什么选择天地图？

| 特性 | 优势 |
|------|------|
| 卫星精度 | 0.2-0.5米分辨率（中国区域最高） |
| 数据更新 | 更新频率高，最新影像 |
| 国内访问 | 速度快，稳定 |
| 官方授权 | 国家测绘地理信息局官方 |

## 🔑 获取天地图API Key

### 1. 注册账号
访问：http://lbs.tianditu.gov.cn/

### 2. 创建应用
1. 登录后进入「控制台」
2. 点击「创建应用」
3. 选择「浏览器端」
4. 填写应用名称，提交

### 3. 获取Key
创建后会获得：
- **Key** (API密钥)
- 免费额度充足

## 📊 天地图图层类型

| 图层名称 | 说明 |
|---------|------|
| `vec` | 矢量底图 |
| `cva` | 矢量注记 |
| `img` | 卫星影像 |
| `cia` | 卫星注记 |
| `ter` | 地形图 |
| `cta` | 地形注记 |

## 🌍 坐标系说明

- 天地图使用 **CGCS2000/WGS84**
- 高德地图使用 **GCJ02**
- 需要坐标转换！
