##  如果地图仍然不显示

### 方案 A：使用备用 CDN

如果 jsDelivr 也无法访问，可以尝试其他 CDN：

#### 修改 `dashboard_leaflet.html`：
```html
<!-- 使用 cdnjs -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/leaflet/1.9.4/leaflet.min.css" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/leaflet/1.9.4/leaflet.min.js"></script>
```

或

```html
<!-- 使用 Staticfile CDN（国内） -->
<link rel="stylesheet" href="https://cdn.staticfile.org/leaflet/1.9.4/leaflet.css" />
<script src="https://cdn.staticfile.org/leaflet/1.9.4/leaflet.min.js"></script>
```

---

### 方案 B：使用百度地图（需要 API Key）

1. 按照 `docs/百度地图配置指南.md` 申请 API Key
2. 修改路由使用百度地图模板：
```python
@equipment_bp.route('/dashboard')
@login_required
def dashboard():
    # ... 代码不变 ...
    return render_template('equipment/dashboard.html', stats=stats)  # 使用百度地图版本
```

---

### 方案 C：本地部署 Leaflet

1. 下载 Leaflet 文件：
   - https://leafletjs.com/download.html
2. 保存到 `app/static/libs/leaflet/`
3. 修改引用：
```html
<link rel="stylesheet" href="{{ url_for('static', filename='libs/leaflet/leaflet.css') }}" />
<script src="{{ url_for('static', filename='libs/leaflet/leaflet.js') }}"></script>
```

## 🆘 常见问题

### Q1: 地图显示灰色方块？
**A**: CDN 资源加载失败
- 检查网络连接
- 打开 Console 查看具体错误
- 尝试更换 CDN 源

### Q2: 侧边栏菜单无法展开？
**A**: JavaScript 未执行或 CSS 问题
- 检查浏览器 Console 是否有 JavaScript 错误
- 确认 base.html 中的 JavaScript 代码完整
- 清除浏览器缓存后刷新

### Q3: 子菜单显示了但无法点击？
**A**: 可能是 z-index 层级问题
- 检查 CSS 中 `.submenu` 的 `z-index` 设置
- 确保子菜单在其他元素之上

### Q4: 设备标记不显示？
**A**: 数据加载失败
- 访问 `/equipment/api/locations` 检查数据
- 确认数据库中有经纬度数据
- 检查浏览器 Console 是否有错误
