## 🚀 快速实施步骤

### 方法一：使用通用 API（最快）

适用于标准列表，只需 3 步：

#### 1. 创建 API 路由（如果还没有）
```python
# 已在 app/__init__.py 中注册
from app.routes.list_api import list_api
app.register_blueprint(list_api)
```

#### 2. 修改列表页面，添加搜索组件
```html
{% extends 'base.html' %}

{% block content %}
<!-- 搜索面板 -->
{% include 'components/list_search_panel.html' %}

<!-- 列表 -->
<table class="table">
    ...
</table>

<!-- 分页 -->
<nav>...</nav>
{% endblock %}
```

#### 3. 修改后端路由，支持搜索参数
```python
keyword = request.args.get('keyword', '').strip()
if keyword:
    query = query.filter(
        Model.name.like(f'%{keyword}%')
    )
```

### 方法二：创建专用 API（推荐用于复杂列表）

#### 1. 创建 API 文件
```python
# app/routes/my_module_api.py
from flask import Blueprint, request, jsonify
from flask_login import login_required

my_module_api = Blueprint('my_module_api', __name__, url_prefix='/api/my-module')

@my_module_api.route('/list')
@login_required
def get_list():
    page = request.args.get('page', 1, type=int)
    per_page = request.args.get('per_page', 20, type=int)
    keyword = request.args.get('keyword', '').strip()
    
    # 构建查询
    query = MyModel.query
    if keyword:
        query = query.filter(
            db.or_(
                MyModel.name.like(f'%{keyword}%'),
                MyModel.code.like(f'%{keyword}%')
            )
        )
    
    pagination = query.order_by(MyModel.id.desc()).paginate(
        page=page, per_page=per_page, error_out=False
    )
    
    return jsonify({
        'code': 200,
        'data': {
            'list': [...],
            'total': pagination.total,
            'page': page,
            'per_page': per_page
        }
    })
```

#### 2. 注册蓝图
```python
# app/__init__.py
from app.routes.my_module_api import my_module_api
app.register_blueprint(my_module_api)
```
