## 📝 使用说明

### 1. 使用批量导出功能
1. 在备件列表页面，勾选需要导出的记录
2. 点击右上角"导出数据"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

### 2. 使用视图切换功能
1. 点击列表页面右上角的网格图标
2. 在列表视图和卡片视图之间切换
3. 选择最适合的查看方式

### 3. 使用库存预警功能
- 🔴 红色感叹号：库存不足（低于最低库存）
- 🟡 黄色感叹号：库存过剩（超过最高库存）
- 🟢 绿色对勾：库存正常
