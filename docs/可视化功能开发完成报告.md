## 三、技术实现细节

### 3.1 仓库平面图
**渲染方式**: DOM + CSS Grid
**核心特性**:
- 按库区分组展示
- 动态生成库位网格
- CSS transition 实现平滑过渡
- 事件委托处理点击

**颜色方案**:
```javascript
空闲：#E0E0E0 (灰色)
占用：#4CAF50 (绿色)
锁定：#FF9800 (橙色)
损坏：#F44336 (红色)
```

### 3.2 库存热力图
**渲染方式**: Canvas 2D
**核心算法**:
```javascript
// 颜色计算（HSL 空间）
const hue = (1 - ratio) * 120; // 绿->红
return `hsl(${hue}, 70%, 50%)`;
```

**维度说明**:
- **密度**: 库存数量分布
- **周转率**: 30/库龄（简化计算）
- **库龄**: 库存天数

### 3.3 入库动画
**渲染方式**: DOM + CSS Animation
**动画步骤**:
1. 入库口出现 (1000ms)
2. 移动到库区 (1500ms)
3. 移动到货架 (1000ms)
4. 放入库位 (1500ms)
5. 库位高亮 (2000ms)

**总时长**: 约 7 秒/物品
