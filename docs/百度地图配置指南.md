# 百度地图 API 配置指南

## 📍 第一步：获取百度地图 API Key

### 1. 访问百度地图开放平台
打开网址：https://lbsyun.baidu.com/

### 2. 注册/登录账号
- 使用百度账号登录（如果没有需要先注册）

### 3. 进入控制台
- 点击右上角 "控制台"

### 4. 创建应用
- 点击 "应用管理" → "我的应用"
- 点击 "+ 创建应用"

### 5. 填写应用信息
- **应用名称**: 备件管理系统（或任意名称）
- **应用类型**: 浏览器端
- **Referer 白名单**: 
  - 开发环境：`localhost`
  - 生产环境：填写您的域名
  - 或填写 `*`（不推荐，仅限测试）

### 6. 获取 AK (API Key)
- 创建成功后，会显示一串字符，这就是您的 API Key
- 格式类似：`6xT8zK9qY3mL5nP2wR7tU1vX4sA0bC`

---

## 🔧 第二步：配置到系统中

### 方法一：修改 base.html（推荐）

1. 打开文件：`app/templates/base.html`

2. 找到以下代码（大约在第 1280 行）：
```html
<!-- 百度地图 API -->
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak=YOUR_API_KEY"></script>
```

3. 将 `YOUR_API_KEY` 替换为您的实际 API Key：
```html
<!-- 百度地图 API -->
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak=6xT8zK9qY3mL5nP2wR7tU1vX4sA0bC"></script>
```

4. 保存文件，刷新页面即可

---

### 方法二：使用配置文件（适合多环境）

1. 在 `app/config.py` 中添加配置：
```python
class Config:
    # ... 其他配置 ...
    Baidu_MAP_API_KEY = os.environ.get('BAIDU_MAP_API_KEY', 'YOUR_API_KEY')
```

2. 在 `.env` 文件中设置环境变量：
```
BAIDU_MAP_API_KEY=6xT8zK9qY3mL5nP2wR7tU1vX4sA0bC
```

3. 在模板中使用：
```html
<script type="text/javascript" src="https://api.map.baidu.com/api?v=3.0&ak={{ config.BAIDU_MAP_API_KEY }}"></script>
```

## ✅ 第三步：验证配置

### 1. 访问设备仪表盘
- 访问：http://localhost:5000/equipment/dashboard

### 2. 检查地图显示
- 应该能看到中国地图
- 13 台设备的标记点应该显示在地图上
- 点击标记点可以查看设备详情

### 3. 如果地图不显示
- 打开浏览器开发者工具（F12）
- 查看 Console 是否有错误信息
- 常见错误：
  - "AK 不存在" → 检查 API Key 是否正确
  - "Referer 校验失败" → 检查 Referer 白名单设置
  - "服务未开启" → 检查是否启用了"浏览器端"服务

## 📊 功能说明

### 已实现功能

1. **设备分布地图**
   - 显示所有有位置信息的设备
   - 不同颜色标记表示不同状态：
     - 🟢 绿色：运行中
     - ⚫ 灰色：停机
     - 🟠 橙色：维修中
     - 🔴 红色：已报废

2. **设备统计卡片**
   - 设备总数
   - 各状态设备数量
   - 有位置信息的设备数量

3. **设备系列统计**
   - 按 FS/MT/CS/MB/FG 系列分类统计

4. **设备列表**
   - 右侧显示设备列表
   - 点击可快速定位到地图上的设备

5. **设备详情弹窗**
   - 点击地图标记查看设备详情
   - 可直接跳转到设备详情页

### 后续可扩展功能

1. **设备选点**
   - 在创建/编辑设备时，通过地图选点
   - 自动获取经纬度和地址

2. **地址解析**
   - 输入地址自动定位
   - 经纬度反向解析为地址

3. **搜索附近设备**
   - 基于当前位置搜索半径 X 公里内的设备

4. **设备分布热力图**
   - 显示设备密度分布

5. **路线规划**
   - 从当前位置到设备位置的导航

## 📝 注意事项

1. **API Key 安全**
   - 不要将 API Key 提交到公共代码仓库
   - 建议使用环境变量管理
   - 生产环境设置 Referer 白名单

2. **配额限制**
   - 百度地图个人开发者：30 万次/天
   - 如果超出配额，需要升级或等待次日重置

3. **合规使用**
   - 遵守百度地图开放平台服务条款
   - 不要用于违法用途
   - 注意用户隐私保护

## 💡 常见问题

### Q1: 地图显示空白？
**A**: 检查以下几点：
1. API Key 是否正确
2. 浏览器 Console 是否有错误
3. 网络是否能访问百度地图 API
4. Referer 白名单是否包含当前域名

### Q2: 标记点不显示？
**A**: 
1. 检查设备是否有经纬度数据
2. 访问 `/equipment/api/locations` 查看数据
3. 确认数据库中的 latitude/longitude 字段有值

### Q3: 如何修改设备的安装位置？
**A**: 
1. 目前可以通过编辑设备，手动修改经纬度
2. 后续会添加地图选点功能
3. 或运行 `scripts/init_device_locations.py` 重新生成位置
