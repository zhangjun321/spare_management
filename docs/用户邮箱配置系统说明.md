# 用户邮箱配置系统

## 功能概述

重新设计的邮箱配置系统，支持为每个系统管理员配置独立的邮箱，实现个性化邮件通知。

## 主要特性

### 1. 个人邮箱配置
- 每个系统管理员可以配置自己的邮箱
- 支持不同的 SMTP 服务器配置
- 独立的授权码管理
- 密码框防自动填充设计

### 2. 智能邮件发送
- 发送预警邮件时，使用管理员自己的邮箱配置
- 如果管理员未配置邮箱，使用系统默认邮箱配置
- 优先级：个人配置 > 系统默认配置

### 3. 测试功能
- 测试连接：仅测试 SMTP 服务器连接
- 发送测试邮件：连接成功后发送测试邮件
- 智能密码检测：自动识别浏览器自动填充的旧密码

## 访问入口

### 个人邮箱配置
- **路径**：用户菜单 → 我的邮箱配置
- **URL**：`/user_email/my-email-config`
- **权限**：所有系统管理员

### 管理员邮箱管理
- **路径**：系统管理 → 管理员邮箱管理
- **URL**：`/user_email/admin/email-configs`
- **权限**：系统管理员

## 使用流程

### 管理员配置个人邮箱

1. 点击右上角用户菜单
2. 选择"我的邮箱配置"
3. 填写 SMTP 配置信息：
   - SMTP 服务器（如：smtp.qq.com）
   - SMTP 端口（如：465）
   - 用户名（完整的邮箱地址）
   - 密码/授权码
   - 发件人邮箱
   - 发件人名称
4. 点击"测试连接"验证配置
5. 点击"发送测试邮件"接收测试邮件
6. 点击"保存配置"保存设置

### 系统发送预警邮件

1. 库存预警触发
2. 系统遍历所有管理员
3. 对每个管理员：
   - 查询其个人邮箱配置
   - 如果有个人配置，使用个人配置发送
   - 如果没有，使用系统默认配置发送
4. 记录发送日志

### 核心代码

**邮件服务优先级逻辑**：

```python
def get_config(self, user_id=None):
    """获取邮件配置"""
    if self.config is None:
        # 优先使用用户邮箱配置
        if user_id:
            self.config = UserEmailConfig.query.filter_by(
                user_id=user_id, 
                is_active=True
            ).first()
        
        # 如果没有用户配置，使用系统默认配置
        if not self.config:
            self.config = EmailConfig.query.filter_by(is_active=True).first()
    
    return self.config
```

## 优势

### 1. 个性化管理
- 每个管理员使用自己的邮箱
- 避免邮件被误判为垃圾邮件
- 提高邮件到达率

### 2. 灵活性
- 支持多个 SMTP 服务器
- 管理员可以随时修改配置
- 不影响其他管理员

### 3. 安全性
- 密码框防自动填充
- 智能识别旧密码
- 优先使用数据库中的正确配置

### 4. 容错性
- 个人配置失败时使用系统默认配置
- 详细的错误日志
- 友好的错误提示

## 常见问题

### Q: 为什么要使用个人邮箱配置？
A: 
- 避免系统邮箱被大量预警邮件标记为垃圾邮件
- 提高邮件送达率
- 管理员可以自定义发件人名称

### Q: 如果管理员没有配置邮箱怎么办？
A: 系统会自动使用默认的系统邮箱配置。

### Q: 如何查看所有管理员的邮箱配置状态？
A: 访问"系统管理 → 管理员邮箱管理"页面。

### Q: 密码框为什么是空的？
A: 为了安全考虑，密码框不会自动填充，需要手动输入。
