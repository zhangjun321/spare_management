## 🔧 使用说明

### 1. 自定义联系方式
修改 `base.html` 中 Footer 部分的联系信息：
```html
<p><i class="fas fa-phone"></i>您的电话号码</p>
<p><i class="fas fa-envelope"></i>您的邮箱地址</p>
<p><i class="fas fa-clock"></i>工作时间</p>
<p><i class="fas fa-map-marker-alt"></i>公司地址</p>
```

### 2. 自定义社交媒体链接
修改社交媒体图标链接：
```html
<a href="您的 GitHub 地址" title="GitHub"><i class="fab fa-github"></i></a>
<a href="您的微信二维码链接" title="微信"><i class="fab fa-weixin"></i></a>
<a href="您的微博地址" title="微博"><i class="fab fa-weibo"></i></a>
<a href="您的 QQ 链接" title="QQ"><i class="fab fa-qq"></i></a>
```

### 3. 添加更多链接
在对应区域添加更多链接：
```html
<li><a href="链接地址"><i class="fas fa-angle-right"></i>链接文字</a></li>
```
