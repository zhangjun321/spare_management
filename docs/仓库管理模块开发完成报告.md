## 🎯 混合模式实现

### 人工录入 + AI 推荐

系统实现了**人工录入与 AI 推荐相结合**的混合模式：

#### 工作流程
```
1. 用户填写备件基本信息
   ↓
2. 点击"AI 推荐"按钮（可选）
   ↓
3. AI 分析备件特性
   - 名称、规格、分类
   - 尺寸、重量
   - 存储要求
   - 出入库频率
   - 价值等级
   ↓
4. AI 返回推荐结果
   - 推荐仓库
   - 推荐货位
   - 置信度
   - 推荐理由
   - 备选方案
   ↓
5. 用户选择：
   - ✅ 接受推荐 → 自动填入
   - ✏️ 修改推荐 → 手动调整
   - ❌ 忽略推荐 → 完全手动
   ↓
6. 保存备件信息
```

#### AI 推荐示例
```json
{
  "success": true,
  "recommended_warehouse_id": 1,
  "recommended_warehouse_name": "北京一号仓库",
  "recommended_location_id": 10,
  "recommended_location_code": "A-01-02-03",
  "confidence": 92,
  "reason": "该仓库专门存储轴承类备件，货位靠近出口，便于高频拣货",
  "alternative_warehouses": [2, 3],
  "notes": "建议定期检查润滑状态"
}
```
