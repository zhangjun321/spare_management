# 仪表盘布局优化报告 - 图表独立 + 数据卡片 3 列布局

## 📋 优化背景

### 用户需求
- ✅ 图表单独布局，独立一行
- ✅ 库存预警、最近交易、待处理维修工单统一布局，放在一行（3 列）
- ✅ 方便后期增加更多图表

### 优化前问题
- ❌ 图表和数据卡片混在一起（2×2 布局）
- ❌ 不方便扩展更多图表
- ❌ 布局不够清晰

## ✅ 优化方案

### 新布局结构

```
┌─────────────────────────────────────────────────────┐
│  统计卡片（4 个，每个 3 列）                          │
├─────────────────────────────────────────────────────┤
│  图表区域（12 列，单独一行）                          │
│  - 库存状态分布饼图                                  │
│  - 后期可添加：趋势图、对比图等                      │
├─────────────────────────────────────────────────────┤
│  库存预警 │ 最近交易 │ 待处理工单（各 4 列）         │
└─────────────────────────────────────────────────────┘
```


## 📱 响应式优化

### 断点设置

#### 大屏（≥992px）
- 图表：12 列（100% 宽度）
- 数据卡片：3 列 × 4 列（33.33% 宽度）

#### 中屏（768px-991px）
- 图表：12 列（100% 宽度）
- 数据卡片：2 列 × 6 列（50% 宽度）
- 第三个卡片自动换行

#### 小屏（≤767px）
- 图表：12 列（100% 宽度）
- 数据卡片：1 列 × 12 列（100% 宽度）
- 所有卡片垂直排列

### 响应式代码

```html
<!-- 使用 col-lg-4 实现响应式 -->
<div class="col-lg-4 col-md-6 col-12 mb-4">
    <!-- 卡片内容 -->
</div>
```

## 🔧 技术实现

### CSS 类使用

#### Bootstrap 类
- `col-12`：12 列（100% 宽度）
- `col-lg-4`：大屏 4 列（33.33%）
- `mb-4`：底部间距（24px）
- `h-100`：高度 100%
- `d-flex`：Flexbox 布局
- `align-items-center`：垂直居中
- `justify-content-center`：水平居中

#### 内联样式
```css
/* 图表区域 */
min-height: 300px;  /* 最小高度 */
max-width: 320px;   /* 图表最大宽度 */

/* 数据卡片 */
max-height: 250px;  /* 最大高度 */
overflow-y: auto;   /* 超出滚动 */
height: 180px;      /* 空状态高度 */

/* 文字截断 */
max-width: 160px;   /* 库存预警 */
max-width: 70px;    /* 最近交易 */
max-width: 140px;   /* 待处理工单 */
```

## 🎯 最佳实践

### ✅ 推荐做法
1. 图表独立布局，方便扩展
2. 数据卡片 3 列统一布局
3. 使用响应式类（col-lg-4）
4. 设置最小/最大高度
5. 文字截断防止溢出
6. 添加数量徽章
7. 优化空状态显示
8. 保持统一样式

### ❌ 避免做法
1. 图表和卡片混合布局
2. 固定宽度（使用百分比）
3. 高度不一致
4. 缺少滚动条
5. 内容溢出不处理
6. 空状态简单文字
