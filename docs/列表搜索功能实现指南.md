## 🚀 如何使用

### 方法一：使用通用列表 API（推荐）

适用于所有标准列表页面。

#### 1. 在 Vue 组件中使用

```vue
<template>
    <div id="myListComponent">
        <!-- 搜索和列表 -->
        <el-card>
            <!-- 搜索表单 -->
            <el-form :inline="true" :model="searchForm">
                <el-form-item>
                    <el-input v-model="searchForm.keyword" 
                              placeholder="搜索..."
                              @keyup.enter="handleSearch">
                    </el-input>
                </el-form-item>
                <el-form-item>
                    <el-button type="primary" @click="handleSearch">搜索</el-button>
                    <el-button @click="handleReset">重置</el-button>
                </el-form-item>
            </el-form>
            
            <!-- 数据表格 -->
            <el-table :data="tableData" v-loading="loading">
                <!-- 列定义 -->
            </el-table>
            
            <!-- 分页 -->
            <el-pagination ... />
        </el-card>
    </div>
</template>

<script>
export default {
    data() {
        return {
            searchForm: {
                keyword: ''
            },
            tableData: [],
            loading: false,
            pagination: {
                page: 1,
                pageSize: 20,
                total: 0
            }
        };
    },
    
    methods: {
        async fetchList() {
            this.loading = true;
            try {
                const params = new URLSearchParams({
                    page: this.pagination.page,
                    per_page: this.pagination.pageSize,
                    keyword: this.searchForm.keyword
                });
                
                // 使用通用列表 API
                const response = await fetch(`/api/list/warehouses?${params}`);
                const result = await response.json();
                
                if (result.code === 200) {
                    this.tableData = result.data.list;
                    this.pagination.total = result.data.total;
                }
            } catch (error) {
                console.error('获取列表失败:', error);
            } finally {
                this.loading = false;
            }
        },
        
        handleSearch() {
            this.pagination.page = 1;
            this.fetchList();
        },
        
        handleReset() {
            this.searchForm.keyword = '';
            this.handleSearch();
        },
        
        handlePageChange(page) {
            this.pagination.page = page;
            this.fetchList();
        }
    },
    
    mounted() {
        this.fetchList();
    }
};
</script>
```

### 方法二：使用专用 API

适用于需要自定义搜索逻辑的列表。

#### 1. 创建专用 API

```python
# app/routes/my_module_api.py
from flask import Blueprint, request, jsonify
from flask_login import login_required
from app.models.my_model import MyModel
from app.extensions import db

my_module_api = Blueprint('my_module_api', __name__, url_prefix='/api/my-module')

@my_module_api.route('/list')
@login_required
def get_list():
    page = request.args.get('page', 1, type=int)
    per_page = request.args.get('per_page', 20, type=int)
    keyword = request.args.get('keyword', '').strip()
    
    query = MyModel.query
    
    if keyword:
        query = query.filter(
            db.or_(
                MyModel.name.like(f'%{keyword}%'),
                MyModel.code.like(f'%{keyword}%')
            )
        )
    
    pagination = query.order_by(MyModel.id.desc()).paginate(
        page=page, per_page=per_page, error_out=False
    )
    
    return jsonify({
        'code': 200,
        'data': {
            'list': [...],
            'total': pagination.total,
            'page': page,
            'per_page': per_page
        }
    })
```

#### 2. 注册蓝图

```python
# app/__init__.py
from app.routes.my_module_api import my_module_api
app.register_blueprint(my_module_api)
```

#### 3. 前端调用

```javascript
const response = await fetch(`/api/my-module/list?page=1&keyword=test`);
```

## 📊 API 接口说明

### 通用列表 API

**URL**: `/api/list/<model_name>`

**参数**:
- `page` (int): 页码，默认 1
- `per_page` (int): 每页数量，默认 20
- `keyword` (string): 关键词
- `sort_field` (string): 排序字段
- `sort_order` (string): 排序方式 (asc/desc)

**响应**:
```json
{
    "code": 200,
    "message": "success",
    "data": {
        "list": [...],
        "total": 100,
        "page": 1,
        "per_page": 20,
        "pages": 5
    }
}
```

### 备件专用 API

**URL**: `/api/spare-parts/list`

**参数**: 同通用列表 API

**响应**: 包含备件特有字段（category_name, supplier_name 等）
