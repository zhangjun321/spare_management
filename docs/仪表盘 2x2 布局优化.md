# 仪表盘 2×2 布局优化报告

## 📋 优化背景

### 原始问题
- ❌ 库存状态分布卡片过大（占 8 列），浪费空间
- ❌ 库存预警卡片过小（占 4 列），内容显示不完整
- ❌ 布局不协调，视觉不平衡
- ❌ 卡片高度不一致

### 优化目标
- ✅ 改为 2×2 网格布局，每个卡片占 6 列
- ✅ 统一卡片高度，视觉更协调
- ✅ 优化内容显示，提升可读性
- ✅ 增强交互体验

## ✅ 优化方案

### 1. 布局结构调整

#### 优化前布局
```
┌─────────────────────────────────────────────────────┐
│  统计卡片（4 个，每个 3 列）                          │
├─────────────────────────────────────────────────────┤
│  库存状态分布（8 列）  │  库存预警（4 列）            │
├─────────────────────────────────────────────────────┤
│  最近交易（6 列）      │  待处理工单（6 列）          │
└─────────────────────────────────────────────────────┘
```

#### 优化后布局
```
┌─────────────────────────────────────────────────────┐
│  统计卡片（4 个，每个 3 列）                          │
├─────────────────────────────────────────────────────┤
│  库存状态分布（6 列）  │  库存预警（6 列）            │
├─────────────────────────────────────────────────────┤
│  最近交易（6 列）      │  待处理工单（6 列）          │
└─────────────────────────────────────────────────────┘
```

### 2. 具体优化内容

#### 2.1 库存状态分布卡片

**优化内容**：
- ✅ 列数：8 列 → 6 列
- ✅ 高度：固定高度 → 自适应高度（h-100）
- ✅ 图表：居中显示，限制最大宽度 280px
- ✅ 内边距：1.25rem → 1rem var(--spacing-md)

**代码变化**：
```html
<!-- 优化前 -->
<div class="col-md-8 mb-4">
    <div class="card">
        <div class="card-body">
            <canvas id="stockStatusChart" height="100"></canvas>
        </div>
    </div>
</div>

<!-- 优化后 -->
<div class="col-lg-6 mb-4">
    <div class="card h-100">
        <div class="card-body d-flex align-items-center justify-content-center">
            <div style="width: 100%; max-width: 280px;">
                <canvas id="stockStatusChart"></canvas>
            </div>
        </div>
    </div>
</div>
```

#### 2.2 库存预警卡片

**优化内容**：
- ✅ 列数：4 列 → 6 列
- ✅ 高度：固定 → 自适应（h-100）
- ✅ 最大高度：220px，超出滚动
- ✅ 标题：添加数量徽章
- ✅ 内容：文字截断，防止溢出
- ✅ 空状态：优化显示（图标 + 文字）

**代码变化**：
```html
<!-- 优化前 -->
<div class="col-md-4 mb-4">
    <div class="card">
        <div class="card-header">库存预警</div>
        <div class="card-body p-0">
            <!-- 内容 -->
        </div>
    </div>
</div>

<!-- 优化后 -->
<div class="col-lg-6 mb-4">
    <div class="card h-100">
        <div class="card-header">
            库存预警 <span class="badge bg-warning ms-2">{{ low_stock_parts|length }}</span>
        </div>
        <div class="card-body p-0" style="max-height: 220px; overflow-y: auto;">
            <!-- 内容 -->
        </div>
    </div>
</div>
```

#### 2.3 最近交易卡片

**优化内容**：
- ✅ 高度：自适应（h-100）
- ✅ 最大高度：220px，超出滚动
- ✅ 表格：简化列名（交易编号 → 编号）
- ✅ 编号：显示后 8 位
- ✅ 备件名：文字截断（max-width: 80px）
- ✅ 字体：小号显示
- ✅ 表头：固定顶部（sticky-top）
- ✅ 空状态：优化显示

**代码变化**：
```html
<!-- 优化前 -->
<td>{{ transaction.transaction_number }}</td>
<td>{{ transaction.spare_part.name }}</td>

<!-- 优化后 -->
<td><small>{{ transaction.transaction_number[-8:] }}</small></td>
<td><small class="text-truncate d-inline-block" style="max-width: 80px;">{{ transaction.spare_part.name }}</small></td>
```

#### 2.4 待处理维修工单卡片

**优化内容**：
- ✅ 高度：自适应（h-100）
- ✅ 最大高度：220px，超出滚动
- ✅ 标题：添加数量徽章
- ✅ 工单号：文字截断（max-width: 160px）
- ✅ 徽章：固定宽度（flex-shrink-0）
- ✅ 空状态：优化显示

**代码变化**：
```html
<!-- 优化前 -->
<div class="card-header">待处理维修工单</div>

<!-- 优化后 -->
<div class="card-header">
    待处理维修工单 <span class="badge bg-warning ms-2">{{ pending_maintenance|length }}</span>
</div>
```

### 3. Chart.js 图表优化

#### 优化内容
- ✅ 边框：添加白色边框（2px）
- ✅ 宽高比：调整为 1.5
- ✅ 图例：
  - 使用圆点样式（pointStyle: 'circle'）
  - 增加内边距（padding: 15）
  - 字体大小：11px
- ✅ 提示框：显示百分比

**代码变化**：
```javascript
// 优化前
options: {
    plugins: {
        legend: {
            position: 'bottom'
        }
    }
}

// 优化后
options: {
    responsive: true,
    maintainAspectRatio: true,
    aspectRatio: 1.5,
    plugins: {
        legend: {
            position: 'bottom',
            labels: {
                padding: 15,
                usePointStyle: true,
                pointStyle: 'circle',
                font: {
                    size: 11
                }
            }
        },
        tooltip: {
            callbacks: {
                label: function(context) {
                    // 显示数量和百分比
                }
            }
        }
    }
}
```

## 📱 响应式优化

### 断点设置
- **大屏（≥992px）**：2×2 布局（每行 2 个）
- **中屏（768px-991px）**：2×2 布局（自动调整）
- **小屏（≤767px）**：1×4 布局（单列显示）

### 代码实现
```html
<div class="col-lg-6 mb-4">
    <!-- 卡片内容 -->
</div>
```
- 使用 `col-lg-6`：大屏 6 列（50%）
- 自动适配：中屏和小屏自动换行

## 🔧 技术实现

### CSS 类使用

#### Bootstrap 类
- `col-lg-6`：响应式列（6 列）
- `mb-4`：底部间距
- `h-100`：高度 100%
- `d-flex`：Flexbox 布局
- `align-items-center`：垂直居中
- `justify-content-center`：水平居中
- `text-truncate`：文字截断
- `overflow-hidden`：溢出隐藏
- `flex-shrink-0`：不收缩
- `sticky-top`：固定顶部
- `table-hover`：悬停效果

#### 内联样式
```css
max-height: 220px;
overflow-y: auto;
max-width: 280px;
max-width: 180px;
max-width: 160px;
max-width: 80px;
```

## 🎯 最佳实践

### ✅ 推荐做法
1. 使用响应式布局（col-lg-6）
2. 统一卡片高度（h-100）
3. 设置最大高度 + 滚动
4. 文字截断防止溢出
5. 添加数量徽章
6. 优化空状态显示
7. 图表居中显示
8. 使用统一间距

### ❌ 避免做法
1. 固定宽度（使用百分比）
2. 内容溢出（不截断）
3. 高度不一致
4. 缺少滚动条
5. 空状态简单文字
6. 图表过大或过小
