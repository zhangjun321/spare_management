# 备件管理系统仪表盘图表设计方案

## 概述

为提升系统的数据可视化能力，在主仪表盘中新增5个Chart.js图表，涵盖备件、设备、维修、出入库等核心模块的统计分析。

---

## 图表设计方案

### 1. 备件出入库趋势图（折线图）
**位置**：第二行图表区
**说明**：展示近30天备件出入库的变化趋势，帮助掌握库存流动规律
**数据维度**：
- X轴：日期（近30天）
- Y轴：数量
- 数据系列：入库、出库
**交互功能**：悬停显示详情

### 2. 维修工单状态分布（环形图）
**位置**：第二行图表区
**说明**：直观展示当前维修工单的状态分布情况
**数据维度**：
- 待处理、进行中、已完成、已取消
**交互功能**：点击可查看该状态下的工单列表

### 3. 备件分类库存价值（水平条形图）
**位置**：第二行图表区
**说明**：按备件分类展示库存价值占比，辅助采购决策
**数据维度**：
- 备件分类名称
- 库存总价值
**交互功能**：悬停显示具体金额

### 4. 设备运行状态趋势（面积图）
**位置**：第三行图表区
**说明**：展示近7天设备运行状态的变化趋势
**数据维度**：
- 日期（近7天）
- 运行中、停机、维修中数量
**交互功能**：点击图例切换显示

### 5. 仓库利用率（柱状图）
**位置**：第三行图表区
**说明**：展示各仓库的容量利用率
**数据维度**：
- 仓库名称
- 容量利用率百分比
**交互功能**：点击可跳转至仓库详情

---

## 页面布局设计

### 调整后的仪表盘布局

**第一行**：6个关键统计卡片（已有）
- 备件种类、设备总数、运行中、待处理维修、库存预警、今日交易

**第二行**：设备分布图（8列） + 维修状态环形图（4列）

**第三行**：备件出入库趋势图（6列） + 备件分类价值图（6列）

**第四行**：设备状态趋势图（6列） + 仓库利用率（6列）

**第五行**：现有3个数据卡片（库存预警、最近交易、待处理维修）

---

## 技术实现要点

1. **后端API**：新增5个数据统计接口
2. **图表样式**：统一设计风格，与系统保持一致
3. **响应式**：适配不同屏幕尺寸
4. **数据更新**：支持手动刷新或定时更新
5. **报表导出**：可考虑后续添加导出功能
