# 使用百度千帆生成入库管理轮播图

## 📋 准备工作

### 1. 获取百度千帆 API 密钥

1. 访问 [百度智能云千帆控制台](https://console.bce.baidu.com/qianfan/)
2. 登录百度智能云账号（没有需要先注册）
3. 进入 **应用管理** -> **创建应用**
4. 选择 **文心一格** 服务
5. 创建完成后，获取 **API Key** 和 **Secret Key**

### 2. 配置 API 密钥

打开脚本文件：`d:\Trae\spare_management\scripts\generate_inbound_images_baidu.py`

找到第 12-13 行，替换为您的密钥：

```python
API_KEY = 'YOUR_API_KEY'  # 替换为您获取的 API Key
SECRET_KEY = 'YOUR_SECRET_KEY'  # 替换为您获取的 Secret Key
```

### 3. 安装依赖

```bash
pip install requests
```

## 🚀 生成图片

### 方式 1：直接运行脚本

```bash
cd d:\Trae\spare_management
python scripts\generate_inbound_images_baidu.py
```

### 方式 2：在 IDE 中运行

1. 在 Trae 或其他 IDE 中打开 `generate_inbound_images_baidu.py`
2. 右键 -> 运行
3. 等待生成完成

## ⚠️ 注意事项

### 1. 分辨率说明
- 百度千帆 SD-XL 模型默认生成 **1024x576** 分辨率
- 这是高清图片，但不是真正的 16K
- 如需 16K 超高清，建议使用 Midjourney

### 2. API 使用限制
- 免费额度：新用户有一定免费额度
- 超出后按量计费：约 0.05 元/张
- 具体价格查看百度智能云官网

### 3. 生成时间
- 每张图片约需 30-60 秒
- 6 张图片总共约 3-5 分钟

### 4. 如果生成失败
- 检查 API 密钥是否正确
- 检查网络连接
- 查看控制台错误信息
- 确认账户余额充足

## 🔧 常见问题

### Q1: 提示"API Key 无效"
**A**: 检查是否正确复制了 API Key，注意不要有多余空格

### Q2: 提示"余额不足"
**A**: 登录百度智能云控制台充值

### Q3: 图片质量不满意
**A**: 可以修改脚本中的 prompt 提示词，添加更多细节描述

### Q4: 想要更高分辨率
**A**: 百度千帆目前最高支持 1024x576，如需 16K 建议使用：
   - Midjourney（推荐）
   - Stable Diffusion（本地部署）
   - DALL-E 3
