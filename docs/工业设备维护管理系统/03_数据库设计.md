| 字段名 | 类型 | 长度 | 允许NULL | 默认值 | 说明 |
|--------|------|------|---------|--------|------|
| id | INT | - | NO | AUTO_INCREMENT | 主键 |
| name | VARCHAR | 100 | NO | - | 类型名称 |

| description | TEXT | - | YES | NULL | 描述 |
