# 工业设备维护管理系统 - UI/UX设计文档

## 1. 设计原则

### 1.1 设计理念
- **工业风格**: 简洁、专业、高效
- **数据优先**: 关键数据突出展示
- **操作便捷**: 降低操作复杂度
- **响应式**: 支持多端访问

### 1.2 设计规范
- **色彩体系**: 以蓝色、深灰为主，红/黄/绿作为状态提示色
- **字体**: 使用无衬线字体（Inter、PingFang SC）
- **间距**: 4/8/16/24px基础间距体系
- **圆角**: 4/8/12px圆角设计
