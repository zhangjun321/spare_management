## 🔧 技术实现

### 前端技术栈
- **Bootstrap 5.3.0** - UI 框架
- **Chart.js 4.4.0** - 图表库
- **Font Awesome 6.4.0** - 图标库
- **原生 JavaScript** - 交互逻辑

### API 接口
- RESTful API 设计
- 统一的 JSON 响应格式
- 登录验证保护
- 完善的错误处理

### 数据流
```
用户操作 → JavaScript 处理 → Fetch API → Flask 后端 → 数据库
          ↓                                          ↓
      更新 UI ← 解析响应 ← JSON 响应 ← 业务逻辑 ← 查询结果
```
