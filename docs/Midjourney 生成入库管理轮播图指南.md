## 🎨 生成步骤

### 步骤 1：打开 Midjourney Bot

在 Discord 中：
1. 找到右侧服务器列表中的 **Midjourney** 图标
2. 点击进入任意 `newbies` 或 `fast` 频道
3. 或者在私信中直接使用 `/imagine` 命令

### 步骤 2：输入生成命令

在聊天框输入：

```
/imagine prompt: [提示词] --ar 16:9 --v 6.0 --q 2 --style raw --s 750
```

### 步骤 3：等待生成

- **约 30-60 秒** 后生成 4 张预览图
- 选择最满意的一张，点击 **U1/U2/U3/U4** 放大
- 或点击 **V1/V2/V3/V4** 重新生成变体

### 步骤 4：放大到高清

1. 点击 **U1-U4** 中的任意一个放大图片
2. 等待放大完成（约 20-30 秒）
3. 右键点击图片，选择 **"另存为"** 下载

### 步骤 5：使用 Upscale 功能（可选）

如果需要更高分辨率：
1. 放大后的图片下方会出现 **Upscale** 选项
2. 点击 **Upscale (2x)** 或 **Upscale (4x)**
3. 等待处理完成，获得超高清版本

## ⚙️ 参数说明

### 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `--ar` | `16:9` | **宽高比**：16:9 适合轮播图横幅 |
| `--v` | `6.0` | **版本**：使用 v6.0，效果最好 |
| `--q` | `2` | **质量**：2 倍质量，更精细 |
| `--style` | `raw` | **风格**：真实摄影风格 |
| `--s` | `750` | **风格化**：750（范围 0-1000） |

### 可选参数

- `--chaos 10`：增加多样性（0-100）
- `--seed 1234`：固定随机种子
- `--no text`：排除文字
- `--weird 0`：关闭怪异风格（默认 0）

## 🎯 生成技巧

### 1. 提示词优化

**好的提示词结构**：
```
[主体描述] + [环境描述] + [摄影风格] + [质量词] + [颜色/氛围] + [技术参数]
```

**示例**：
```
Ultra high resolution photograph of warehouse (主体)
bright natural lighting, organized facility (环境)
professional industrial photography (风格)
hyper realistic, extremely detailed (质量)
blue and white, bright atmosphere (颜色)
--ar 16:9 --v 6.0 --q 2 --style raw (参数)
```

### 2. 获得最佳效果

1. **使用 `--style raw`**：更真实，适合工业场景
2. **设置 `--q 2`**：虽然消耗更多，但质量更好
3. **调整 `--s` 值**：
   - 500-750：平衡艺术和真实
   - 750-1000：更艺术化
   - 250-500：更真实

4. **多生成几次**：同样的提示词，每次结果不同
5. **使用 V 功能**：选择满意的图片，点击 V1-V4 生成变体

### 3. 避免常见问题

❌ **避免**：
- 提示词太长（超过 1500 字符）
- 矛盾的描述（如"明亮"和"黑暗"）
- 过于抽象的概念

✅ **推荐**：
- 具体、清晰的描述
- 使用逗号分隔不同元素
- 先测试简单提示词，再逐步添加细节

## 🆘 常见问题

### Q1: 提示词太长被拒绝怎么办？
**A**: Midjourney 限制 1500 字符，可以：
- 删除重复的形容词
- 简化描述
- 分批次生成

### Q2: 生成的图片有文字怎么办？
**A**: 添加 `--no text` 或在提示词中强调 `no text, no watermark`

### Q3: 颜色太暗怎么办？
**A**: 添加 `bright lighting, well-lit, natural light`

### Q4: 细节不够清晰怎么办？
**A**: 使用 `--q 2` 和 `extremely detailed, maximum sharpness`

### Q5: 如何保持 6 张图片风格一致？
**A**: 
- 使用相同的 `--style raw --s 750`
- 使用相同的 seed 值：`--seed 12345`
- 保持提示词结构一致

### Q6: 生成的图片比例不对？
**A**: 确保添加了 `--ar 16:9`

### Q7: 信用卡支付失败？
**A**: Midjourney 支持：
- Visa/Mastercard
- PayPal（推荐）
- 虚拟信用卡
