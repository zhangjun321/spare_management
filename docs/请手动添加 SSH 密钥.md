## 🔑 添加 SSH 密钥到 GitCode（必须手动完成）

### 步骤：

1. **登录 GitCode**
   - 访问：<https://gitcode.com>
   - 使用您的账号登录
2. **进入 SSH 密钥设置页面**
   - 访问：<https://gitcode.com/profile/keys>
   - 或者：点击右上角头像 → 设置 → SSH 密钥
3. **添加 SSH 公钥**
   - 点击 **"添加 SSH 公钥"** 按钮
   - **标题**：填写 `My Computer` 或 `个人电脑`
   - **公钥内容**：完整复制上面的公钥（从 `ssh-ed25519` 开始到结尾）
   - 点击 **"确定"** 或 **"保存"**
4. **验证配置**（可选）
   ```bash
   ssh -T git@gitcode.com
   ```
   如果看到欢迎信息，说明配置成功！
