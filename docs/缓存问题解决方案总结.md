## 📞 常见问题

### Q1: SQLite 缓存性能如何？

**A**: 对于<100 并发用户完全够用，读取速度 1-5ms。

### Q2: 缓存文件会很大吗？

**A**: 默认 5 分钟过期，自动清理，通常<50MB。

### Q3: 需要手动清理吗？

**A**: 不需要，系统自动清理过期缓存。

### Q4: 可以切换到 Redis 吗？

**A**: 可以，安装 Redis 后修改 .env: `USE_REDIS=true`

### Q5: 数据会丢失吗？

**A**: 不会，SQLite 缓存持久化到磁盘文件。
