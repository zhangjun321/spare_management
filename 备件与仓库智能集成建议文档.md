## 🎯 一、现状分析

### 1.1 当前数据模型现状

**现有数据模型关系：**
```
SparePart（备件表）
├── warehouse_id -&gt; Warehouse（仓库表）
├── location_id -&gt; WarehouseLocation（货位表）
├── batches -&gt; Batch（批次表）
└── transactions -&gt; Transaction（交易表）

Warehouse（仓库表）
├── spare_parts -&gt; 该仓库的所有备件
├── locations -&gt; 该仓库的所有货位
└── batches -&gt; 该仓库的所有批次

WarehouseLocation（货位表）
├── warehouse -&gt; 所属仓库
├── spare_parts -&gt; 该货位的备件
└── batches -&gt; 该货位的批次
```

### 1.2 已有的AI服务现状

✅ 已有 `AIInfoFillService`：
- 使用 Qwen/Qwen2.5-72B-Instruct 模型
- 支持备件信息自动填充
- 需要扩展仓库管理的AI服务

## 🔥 二、核心目标（重点更新）

### 2.1 最高优先级：AI全自动仓库数据创建

**一键完成，零人工操作！**

1. **自动分析现有备件数据**
   - 分析备件ABC分类
   - 分析备件属性（重量、体积、特殊存储要求）
   - 分析备件数量和库存分布

2. **自动生成三级库位体系**
   - ✅ 自动创建库区（WarehouseZone）
   - ✅ 自动创建货架（WarehouseRack）
   - ✅ 自动创建货位（WarehouseLocation）

3. **自动分配备件库位**
   - 根据备件特性智能分配最佳存放位置
   - 自动更新备件的 warehouse_id 和 location_id

4. **可视化展示创建过程**
   - 实时进度展示
   - 创建结果报告
   - 可回滚功能

**具体实现：**

##### 3.2.1.1 分析备件数据

```python
def analyze_spare_parts(self, warehouse_id):
    """
    分析仓库的备件数据，为智能初始化做准备
    
    返回：
    {
        'total_parts': 100,
        'abc_distribution': {'A': 20, 'B': 30, 'C': 50},
        'special_requirements': {
            'cold_storage': 5,
            'hazardous': 3,
            'fragile': 10
        },
        'avg_turnover': 0.65,
        'recommendations': [...]

    """
```

**分析维度：**
1. **ABC分类分析** - 按价值和周转率
2. **特殊属性分析** - 冷藏、危险品、易碎品等
3. **周转率分析** - 出库频率
4. **数量分析** - 计算所需库位数量

---

##### 3.2.1.2 生成库区规划

```python
def generate_zone_plan(self, analysis_result):
    """
    根据分析结果生成库区规划
    
    返回：
    {
        'zones': [
            {
                'zone_code': 'A-QUICK',
                'zone_name': '快拣区',
                'zone_type': 'general',
                'abc_type': 'A',
                'rack_count': 5,
                'locations_per_rack': 20,
                'description': 'A类高周转备件'
            },
            ...
        ],
        'total_racks': 20,
        'total_locations': 200
    }
    """
```

**规划规则：**
1. **A类备件（高价值/高周转）→ 快拣区**
   - 位置：靠近出入口
   - 货架：低层，易存取
   - 库位：快拣专用位

2. **B类备件（中价值/中周转）→ 普通区**
   - 位置：中间区域
   - 货架：标准货架
   - 库位：普通库位

3. **C类备件（低价值/低周转）→ 高架区**
   - 位置：深处或高层
   - 货架：高架货架
   - 库位：密集存储

4. **特殊备件 → 专用库区**
   - 冷藏备件 → 冷藏区
   - 危险品 → 危险品区
   - 易碎品 → 固定底层位
