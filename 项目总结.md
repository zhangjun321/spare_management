## 📋 下一步操作指南

### 立即执行 (必须)

#### 1. 安装 Python 依赖

```bash
cd d:\Trae\spare_management
pip install -r requirements.txt
```

#### 2. 初始化数据库 (三选一)

**方法 A: 使用 MySQL Workbench (推荐)**
1. 打开 MySQL Workbench
2. 连接 MySQL (root / 您的密码)
3. 执行:
```sql
CREATE DATABASE spare_parts_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE spare_parts_db;
-- 然后执行 create_tables.sql 文件内容
```

**方法 B: 使用命令行**
```bash
mysql -uroot -p"您的密码" -e "CREATE DATABASE spare_parts_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
mysql -uroot -p"您的密码" spare_parts_db < d:/Trae/spare_management/scripts/create_tables.sql
```

**方法 C: 使用 Python 脚本**
```bash
# 先修改 .env 文件中的数据库密码
python scripts/init_db.py
```

#### 3. 启动应用测试

```bash
python run.py
```

访问 http://localhost:5000
